🛡️ Incident Response Order

Containment → Preservation → Eradication → Recovery
MEMORIZE: C.P.E.R — Contain first, always

🔒 Phase 1: Containment

  • Disconnect from network
  • Disable compromised accounts
  • Block malicious IPs/domains
  • Segment affected VLANs

💾 Phase 2: Preservation

  • Capture memory/disk images
  • Preserve logs (forward to SIEM)
  • Chain of custody documentation

🧹 Phase 3: Eradication

  • Remove malware/persistence
  • Patch vulnerabilities
  • Reset compromised credentials

🔄 Phase 4: Recovery

  • Restore from clean backups
  • Monitor for re-infection
  • Post-incident review

🎯 Attack Pattern Library

Know the signature → spot it in logs → stop it in production.

💀 Ransomware Malware
Signature: Mass file encryption, .locked/.encrypted, ransom notes
Detection: High file mod rate, vssadmin delete
Defense: Offline backups, EDR, network segmentation
🔄 Reverse Shell C2
Signature: Outbound port 4444/8080, regular beaconing
Detection: netstat -ano, unusual outbound from server
Defense: Egress filtering, IDS signatures
🔑 Brute-force Credential
Signature: High volume Event 4625 from same source
Detection: >10 failed logons in 60s from single IP
Defense: Account lockout, MFA, fail2ban
📂 Directory Traversal Web
Signature: ../../../etc/passwd, %2e%2e%2f in URL
Detection: WAF rules, HTTP 400s with traversal patterns
Defense: Input validation, WAF, chroot
🗄️ SQL Injection Web
Signature: ' OR 1=1-- , UNION SELECT, time-based delays
Detection: WAF, unusual SQL errors in app logs
Defense: Parameterized queries, least privilege DB
🔐 Pass-the-Hash Lateral
Signature: Mimikatz, NTLM hash reuse, Type 3 logons
Detection: Event 4624 Type 3 from workstations
Defense: Credential Guard, tiered admin model
👤 Kerberoasting AD
Signature: TGS requests for multiple SPNs, RC4 encryption
Detection: Event 4769 with RC4_HMAC (0x17)
Defense: Long SPN passwords, AES-only, gMSA
🎣 Phishing Social
Signature: Lookalike domains, urgent language, malicious attachments
Detection: Email gateway, DMARC/SPF/DKIM
Defense: User training, MFA, sandboxing
🌐 DNS Tunneling Exfil
Signature: Long TXT queries, base64 in subdomains, high query volume
Detection: Entropy analysis, DNS request size anomalies
Defense: DNS filtering, Sinkhole, Zscaler
🏔️ Living Off The Land Malware
Signature: powershell.exe, wmic, certutil, bitsadmin abuse
Detection: Process tree analysis, AMSI logs
Defense: Constrained PowerShell, AppLocker

🔗 Common Persistence Mechanisms

  • Run / RunOnce — Registry: HKLM\Software\Microsoft\Windows\CurrentVersion\Run
  • Scheduled Tasks — Event ID 4698
  • Services — Event ID 7045
  • WMI Event Subscription — EventFilter / EventConsumer
  • Startup Folder — %AppData%\Microsoft\Windows\Start Menu\Programs\Startup
  • Boot/Logon Scripts — GPO or local
  • DLL Hijacking — Search order exploitation
  • AppInit_DLLs — Registry: HKLM\Software\Microsoft\Windows NT\CurrentVersion\Windows
🔍 WMI Persistence Example
# Attacker creates a WMI filter that triggers on startup
wmic /NAMESPACE:"\\root\subscription" PATH __EventFilter CREATE Name="MaliciousFilter", EventNameSpace="root\cimv2", QueryLanguage="WQL", Query="SELECT * FROM __InstanceCreationEvent WHERE TargetInstance ISA 'Win32_Process' AND TargetInstance.Name='explorer.exe'"
# Then binds to a consumer (e.g., ActiveScriptEventConsumer) to run code
