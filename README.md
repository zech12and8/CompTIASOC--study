# 🧠 SOC.sheet — CompTIA Security+ / SOC Analyst Study Dashboard

> **Active destruction beats potential destruction — always.**

A comprehensive, interactive HTML5.3+ study dashboard for SOC analysts and CompTIA Security+ candidates. Built with the 2026 CompTIA color scheme, this tool transforms dense exam material into an accessible, hands-on learning experience.

---


## 📋 **What's Inside**

| Section | Content |
|---------|---------|
| **Alert Priority Matrix** | 10 real-world alerts ranked CRITICAL → LOW with investigation reasoning |
| **Incident Response** | C.P.E.R framework (Containment → Preservation → Eradication → Recovery) |
| **Attack Pattern Library** | 10 attack patterns: Ransomware, Reverse Shell, Brute-force, Directory Traversal, SQLi, Pass-the-Hash, Kerberoasting, Phishing, DNS Tunneling, Living Off The Land |
| **Windows Reference** | Logon Types (2-11) + Critical Event IDs (4624, 4625, 4688, 4698, 4720, 4732, 7045, 1102) |
| **Persistence Mechanisms** | Registry, Scheduled Tasks, Services, WMI, Startup Folder, DLL Hijacking |
| **CLI Installations** | One-click commands for Ubuntu/Debian, RHEL/CentOS/Fedora, Arch/Manjaro, Alpine |
| **Practice Terminal** | Click-to-copy commands with sample output (netstat, tcpdump, nmap, tail) |
| **Interactive Quiz** | Test your knowledge — press `R` to reset |
